Este repositório apresenta um projeto fictício de implementação de serviços AWS realizado na empresa Abstergo Industries, conduzido por Thiago Ribeiro.
O objetivo principal foi reduzir custos imediatos, mantendo a eficiência operacionaI e aumentando a escalabilidade da infraestrutura em nuvem.
O projeto foi dividido em três etapas, cada uma focada em um serviço AWS estratégico para economia e otimização de recursos.
